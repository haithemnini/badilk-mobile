# Changelog

## [1.0.2] - 2026-01-01

### Added
- 500+ new device models across all major brands
- Enhanced search filters for better results
- New premium features for subscribers
- Improved dark mode color scheme
- Better error messages and user feedback

### Changed
- Updated UI/UX for home screen
- Improved app performance and loading times
- Enhanced notification system
- Refined search algorithm for more accurate results

### Fixed
- Fixed issue with favorite sync across devices

### Security

- Improved authentication flow
